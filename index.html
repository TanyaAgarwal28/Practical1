About Course

Full Stack development refers to the development of both front end(client side) and back end(server side).It includes various languages as well as various frameworks for frontend and backend. Typically languages used in this course are Hypertext Markup Language(HTML), Cascading style sheets(CSS) and JavaScript. HTML is used to develop static webpages basically it is used to display the content on the web page, CSS is used to style the content on the webpage and Javascript gives the behaviour to the web page i.e make it responsive.We are having various frameworks like react. bootstrap etc for frontend whereas for backend we can use nodejs, Django etc. Languages that can be used for backend are java, ruby, python etc. Overall the main objective of this course is to develop a websites that are responsive, creative and well defined.
